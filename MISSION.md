```

  → 목표: 실전 월 수익률 2%+ 3개월 연속 달성
```

## 절대 잊지 말 것

**모델은 도구다. 판단은 우리가 한다.**

- 모델이 아무리 좋아도 3일 연속 크게 틀리면 → 멈추고 재검토
- 한 달 수익이 -10% 넘어가면 → 무조건 거래 중단, 원인 분석
- 수익에 취해서 포지션 키우지 말 것 → 항상 리스크 2% 규칙 지킴
- TSLL은 레버리지 ETF다 → 한 방에 40~60% 날아갈 수 있음
- 생계가 달린 순간부터는 멘탈 관리가 코드만큼 중요하다
